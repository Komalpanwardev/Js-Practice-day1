# JavaScript Mini Projects

This repository contains beginner-friendly JavaScript projects built while learning core JavaScript concepts.

### 1. Student Bio Generator

### 2. BMI Calculator

A JavaScript project that calculates Body Mass Index using height and weight.

### 3. Age Checker

Checks whether a person is a minor or an adult using conditional statements.

### 4. Type Checker Tool

Detects different JavaScript data types using the `typeof` operator.

Calculates savings based on salary and expenses.
